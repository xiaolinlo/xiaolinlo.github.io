Win10 系统搭建 Hexo 静态博客

发表于 2020-09-11 更新于 2020-09-17

Hexo 博客框架是目前最热门的静态博客框架之一。本文主要介绍如何在 win10 系统上搭建和部署一个属于自己的个人博客。如有错误，欢迎指正。

安装 Node.js

1）下载安装

Node.js是搭建个人博客的必要工具。用于搭建个人博客的文件、程序代码都是依靠它来产生的。Node.js 可以到官网进行下载安装。

Node.js 官方网站：http://nodejs.cn/

下载安装完成后会得到一个 node.js 的软件和一个 npm 包管理器。

2）命令行查看版本
